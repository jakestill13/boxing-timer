# Boxing Timer Constitution

1. **Precision**: Usar o relógio do sistema para evitar drift no cronômetro.
2. **High Visibility**: UI de alto contraste (Vermelho/Verde) para ser lida à distância.
3. **Sound First**: O gongo é o gatilho principal de mudança de estado.
4. **Mobile Ready**: Interface otimizada para ser usada em navegadores de celular/tablet.
